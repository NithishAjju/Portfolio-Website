Projects

NetSentinel: LLM-Powered Security Q&A Bot

Built a secure and intelligent Q&A chatbot designed to assist users with network security-related queries. Leveraging a Retrieval-Augmented Generation (RAG) approach, the bot processes user questions using natural language processing (NLP) and retrieves relevant answers from uploaded PDF resources. The system uses ChromaDB for vector storage, Hugging Face embeddings for semantic understanding, and the Groq API to generate accurate, context-aware responses. A clean web interface built with Flask, HTML, CSS, and JavaScript enables real-time interactions. The chatbot includes features like source traceability, time-stamped messages, and dynamic user feedback, ensuring transparency and an intuitive user experience.

Covid 19 Data Augmentation using GAN's

Built a Generative Adversarial Network (GAN) using PyTorch to generate high-quality synthetic COVID-19 images, with CUDA support for fast, GPU-accelerated training and Torchvision for streamlined image preprocessing. By fine-tuning hyperparameters, I improved the model’s Intersection over Union (IoU) by 15%, significantly enhancing image segmentation accuracy. The generator and discriminator were carefully designed using deep convolutional layers and batch normalization, resulting in stable training and more realistic image outputs.

YouTube Transcript Summarizer

Designed and deployed a Streamlit-based web application that transforms YouTube video transcripts into detailed, context-aware summaries using a fine-tuned LLaMA-2 7B Large Language Model. Leveraging LoRA (Low-Rank Adaptation) for efficient fine-tuning and prompt engineering techniques, the model achieved a 10% improvement in BERT Score, enhancing summarization quality. The app supports customizable summary lengths (50–500 words) and multilingual output, making it a versatile tool for education, research, and content analysis. Built with Python, Streamlit, and the YouTube Transcript API, the project also integrates secure environment management via dotenv.

Predictive Modeling for Flight Booking Forecasting

Built an end-to-end machine learning solution to predict customer booking completions for British Airways using a dataset of 50,000 flight reservations. Engineered 24 insightful features capturing customer behavior, handled class imbalance with SMOTE, and fine-tuned a Random Forest model to hit 90% accuracy. Through visual feature analysis with Seaborn and metrics like ROC-AUC and confusion matrices, I uncovered key booking patterns—like the impact of purchase lead time and trip type. The final model offered actionable insights to help British Airways optimize marketing strategies and operational planning.

PhishShield-Scalable ML-Based Phishing Detection

PhishShield is a scalable machine learning-based phishing detection system designed to identify malicious websites with 96% accuracy. Built using Python and Scikit-learn, the project features a complete ML pipeline including data preprocessing, model training, validation, and deployment. It leverages AWS S3 for storage and EC2 for compute, with MLflow integrated for experiment tracking and model versioning. The pipeline is containerized using Docker to ensure portability across environments, and GitHub Actions automate CI/CD workflows for seamless model updates. This project showcases my ability to develop secure, production-ready ML solutions with strong MLOps integration.
